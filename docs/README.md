This folder contains reference documentation supporting the Medicare Integrity
Decision Support framework, including data source documentation, methodology
references, and CMS organizational reference materials.

## Contents

Documentation will be added here as modules are completed. Expected contents
include:

- Data dictionary references for CMS Medicare Part B Public Use File
- CMS organizational structure reference materials
- Methodology and technical specification documents
- Analytical framework reference guides
